# Функции в Python

Функция - это подпрограмма с именем, которая выполняет задачу, принимает аргументы и может возвращать результат

```python
def имя(параметры):
    return результат
```

Вызов функции:

```python
x = имя(значения)
```

## Основные свойства

| Свойство               | Описание                             |
| ---------------------- | ------------------------------------ |
| Переиспользование кода | функцию можно вызывать много раз     |
| Параметры              | входные данные в `(...)`             |
| Возврат значения       | `return` отправляет результат наружу |
| Тело функции           | код, написанный с отступом после `:` |

## Проверка входных данных в функциях

Если требуется, то функции должны сами проверять аргументы:

```python
def пример(a, b):
    if a > b:
        return "Error!"
    return "OK"
```

## Возврат сложных структур

Функция может вернуть список, кортеж или матрицу:

```python
def primes(a, b):
    return [2,3,5]

def matrix_add(a, b):
    return [[1,2],[3,4]]
```
