# Лямбда-функции в Python

Лямбда-функция - это **короткая анонимная функция**, записываемая в одну строку

Используется тогда, когда функция нужна один раз и очень простая

## Синтаксис

```python
lambda аргументы: выражение
```

Пример:

```python
lambda x: x * 2
```

Эквивалентно обычной функции:

```python
def f(x):
    return x * 2
```

## Особенности лямбда-функций

* не имеют имени
* состоят из одного выражения
* автоматически возвращают результат
* не используют `return` (в python)
* чаще всего применяются вместе с `sorted`, `map`, `filter`, `max`, `min`

## Лямбда и `sorted`

Лямбда-функция часто используется для указания ключа сортировки:

```python
sorted(data, key=lambda x: x[1])
```

Здесь `lambda` определяет, по какому полю выполняется сортировка

## Лямбда и `map`

`map` применяет функцию ко всем элементам последовательности:

```python
map(lambda x: x * 2, numbers)
```

Результатом является новый набор значений, полученных после применения функции

## Лямбда и `filter`

`filter` оставляет только те элементы, для которых функция возвращает `True`:

```python
filter(lambda x: x > 10, numbers)
```

Используется для отбора данных по условию

## Лямбда и `max` / `min`

С помощью `lambda` можно указать правило сравнения элементов:

```python
max(items, key=lambda x: x["score"])
```

Это позволяет находить максимальный или минимальный элемент по нужному критерию

## Когда НЕ стоит использовать лямбда-функции

* если логика сложная
* если нужно несколько условий
* если функция используется многократно
* если код становится плохо читаемым

В таких случаях лучше использовать обычную функцию `def`
